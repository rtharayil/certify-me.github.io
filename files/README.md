# Files Directory

This directory is for storing downloadable files such as:
- CV/Resume PDFs
- Certificates
- Documents
- Other files

## Usage

Place your files in this directory and reference them using the `/files/` path in your HTML/Liquid templates.

For example:
- A CV file named `Riya-Sharma-CV.pdf` can be linked as `/files/Riya-Sharma-CV.pdf`
